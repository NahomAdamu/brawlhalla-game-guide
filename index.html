Pygame Fighting Game Guide

Learn how to build a 2D fighting game like Brawlhalla using Python & Pygame

Setup Game Window Player Movement Attacks GitHub

1. Install Python & Pygame

Download Python from python.org.

pip install pygame

2. Create the Game Window

Add the following code in main.py:

import pygame
pygame.init()
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("2D Fighting Game")
running = True
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False
    screen.fill((30, 30, 30))
    pygame.display.update()
pygame.quit()

3. Add a Player

Create an assets/ folder and add a player sprite. Then load and draw it:

player_img = pygame.image.load("assets/player.png")

4. Add Movement

Use arrow keys to move the player:

if keys[pygame.K_LEFT]: player_x -= 5
if keys[pygame.K_RIGHT]: player_x += 5

5. Add Attacks

Press Spacebar to attack:

if keys[pygame.K_SPACE]:  
    player_img = attack_frames[0]  # Attack animation

6. Publish on GitHub

Push your project to GitHub:

git init
